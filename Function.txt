기존 WaveHome 기능을 살린 수면, 자세 교정 케어 시스템

WaveHome Care Mode
1. Desk Mode: 자세교정, 집중 루틴
2. Bed Mode: 수면 준비, 수면 분석
3. Home Control: 가전 제어

자세교정
1. 실시간 상태 관측 (대시보드)
현재 자세 상태
-	좋음
-	주의
-	교정 필요
-	휴식 필요
현재 감지된 자세
-	목이 앞으로 나감
-	허리가 굽음
-	몸이 왼쪽으로 기울어짐
-	어깨가 말림
-	같은 자세 장시간 유지

2. 알림 기능
1단계: 부드러운 인식 알림 (1단계는 대시보드로만)
자세가 살짝 무너졌지만 오래 지속되진 않았을 때.
-	“목이 조금 앞으로 나왔어요.”
-	“허리가 살짝 말리기 시작했어요.”
-	“지금 자세를 한 번만 다시 세워볼까요?”

2단계: 교정 유도 알림 (라즈베리파이 + USB 스피커로 음성 출력)
나쁜 자세가 일정 시간 지속됐을 때.
-	“거북목 자세가 8분째 유지되고 있어요. 턱을 살짝 당겨주세요.”
-	“상체가 왼쪽으로 기울어진 상태가 반복되고 있어요.”
-	“허리를 펴기보다, 골반을 먼저 세워보세요.”

3단계: 휴식 제안 (라즈베리파이 + USB 스피커로 음성 출력)
자세가 계속 무너지거나 장시간 움직임이 없을 때.
-	“자세를 고치기보다 1분 쉬는 게 더 좋아 보여요.”
-	“50분 이상 같은 자세가 유지됐어요. 목·어깨 리셋 루틴을 추천합니다.”
-	“지금은 자세 교정보다 짧은 기지개가 우선입니다.”

3. 리포트로 자세 패턴을 보여주기
일간 리포트
오늘의 자세 점수
-	82점
-	어제보다 6점 상승
-	오후 시간대 자세 저하 반복
오늘의 자세 요약
-	책상 앞 체류 시간: 5시간 20분
-	바른 자세 유지: 62%
-	거북목 위험 시간: 48분
-	허리 굽음 시간: 1시간 10분
-	가장 자세가 무너진 시간대: 오후 3시~5시
오늘의 한 줄 피드백
-	“오늘은 오래 앉아 있을수록 허리보다 목 자세가 먼저 무너졌어요. 내일은 50분마다 목 스트레칭을 제안할게요.”

주간 리포트
이번 주 변화
-	자세 점수: 74점 → 81점
-	거북목 지속 시간: 18% 감소
-	허리 굽음 빈도: 9% 증가
-	휴식 루틴 수행률: 64%
다음 주 목표
-	50분 착석 후 1분 휴식 4회 달성
-	오후 3시 이후 거북목 지속 시간 20% 감소
-	허리 리셋 루틴 하루 2회 수행

수면 관리
1. 사용자가 수면 전에 제공되는 기능들:
- 스마트폰 사용 감지
- 지금 얼른 잠들지 않으면 수면 부채가 []만큼 증가합니다. (그러면 내일은 어떻게 되고 주말은 어떻게 되고 중얼중얼)
2. 사용자가 수면 중에 제공되는 기능들:
- 수면 자세 변화 감지: 자세 변화를 감지하고, 온도가 높고 사용자가 뒤척이는 경우 에어컨을 조절하는 기능을 제공
- 알람: 인간의 생체 시계(Circadian Rhythm)에 맞춰, 자연스러운 빛과 소리로 사용자를 깨우는 기능을 제공. 예를 들어, 수면 단계에 따라 점점 밝아지는 조명과 새소리, 파도 소리 등을 통해 사용자가 자연스럽게 깨어날 수 있도록 지원(수면 관성 최소화, Dawn Simulation, 심박).
3. 사용자가 아침에 일어났을 때 제공되는 수면 인사이트 정보들:
일간 리포트
- 종합 수면 점수
- 수면 데이터 시각화:
- 수면 단계별 그래프(수면 단계, 호흡, 심박수 등)
- 실내의 온도 변화 그래프(이에 따라 에어컨을 가동했다면 이벤트 정보도 제공)
- 사용자가 밤새 뒤척인 횟수와 시간대별 분석
- 수면 중 호흡 패턴 분석 결과 제공(무호흡증, 코골이 등, 코골이는 별도의 마이크가 필요할 듯)
- 수면 빛 탕감: 현재까지 누적된 수면시간과 이를 해소할 수 있는 주간 플랜 제공
- 종합 수면 인사이트 제공: "최근 3일간 방 온도가 26도 이상으로 올라갈 때마다, 새벽 3시쯤 뒤척임이 평소보다 40% 증가하며 얕은 수면으로 바뀐 것이 관찰되었습니다. 오늘 밤에는 에어컨 예약 설정을 새벽 4시까지로 1시간 늘려보시는 걸 추천합니다"

주간 리포트
이번 주 변화 
    - 점수: 74점 → 81점
    - 총합 수면시간: 18% 감소
    - 등등
다음 주 목표
    - 몇시에 자라
    - 자세 어떻게 해봐라
    - 등등
